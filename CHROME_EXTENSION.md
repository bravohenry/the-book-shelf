# Chrome 扩展安装说明

## 构建扩展

1. 安装依赖：
```bash
npm install
```

2. 构建扩展：
```bash
npm run build:extension
```

构建完成后，所有文件将输出到 `dist` 目录。

## 安装到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 启用右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目中的 `dist` 目录
6. 扩展安装完成后，打开新标签页即可看到书架页面

## 功能说明

- 安装后会自动替换 Chrome 的新标签页
- 所有资源（字体、样式、音频）都已本地化，无需网络连接即可使用
- 使用 localStorage 保存数据，数据存储在本地
- 需要配置 Gemini API Key 才能使用 AI 功能（添加书籍/网站时）

## 注意事项

- 首次安装后，如果之前没有配置过 API Key，会提示输入
- 所有数据存储在浏览器的 localStorage 中
- 如需更新扩展，重新构建后，在扩展管理页面点击刷新按钮即可
